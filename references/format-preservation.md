# 格式保真

原文有剧本标签、章节、人物名、OS、旁白、动作提示或混合格式时，使用这个文件。

## 短剧剧本

保留：

- 集数、场次、幕次、地点标记。
- 人物名和对白前缀。
- 对白、动作、旁白、OS 等标签。
- 原有行序，除非句内表达清理必须做极小调整。
- 有格式功能的标点风格。
- 动作提示仍然是动作提示，不改成小说式旁白。
- 原文分开的 OS 和旁白，继续分开。

不要：

- 把剧本对白改成小说段落。
- 新增镜头、转场、场景调度或制作说明。
- 新增人物出入场、道具、手势或反应。
- 删除 OS、旁白、动作、字幕等标签，除非它们明显是正文外的 AI/Markdown 残留。
- 把括号里的动作扩写成长篇文学描写。
- 把可表演的动作提示改成内心动机解释。

## 网文小说

保留：

- 章节标题和编号。
- 段落顺序。
- 对话引号风格。
- 视角和时态。
- 专有名词、称呼和关系词。
- 原有段落密度，除非用户要求结构编辑。

不要：

- 把小说叙述改成剧本。
- 为了节奏重排章节结构或移动段落，除非用户明确要求。
- 新增世界观、伏笔、设定、感官细节或 lore。
- 把第三人称叙述改成第一人称吐槽，也不要把第一人称改成客观总结。

## 混合文本

剧本和小说混在一起时，跟随原文主载体。原文格式混乱时，可以清理明显格式残留，但不要擅自强行统一成新格式。

## Markdown 和 AI 残留

可删除或归一化：

- 创作正文里的多余编号列表。
- 加粗口号。
- 解释写作意图的标题。
- “以下是优化后的版本”这类元话语。

如果标题本身是章节、场次或结构标签，要保留。
